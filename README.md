# Sarkeon
Sarkeon is a pixel sandbox game, where you explore simulated world left to rot and die by ancient gods, and you are there to take back what is yours

All assets and code is by
n0o0b090lv - https://n0o0b090lv.itch.io/ (also officialErnests)
vivithequeen - https://vivithequeen.itch.io/

Controlls:
Wasd - move
Mouse - look around
Q - place
E - dig
R/F - change inventory slot
